# Reaction-Based-Tile-Game
This project is an interactive browser-based reaction game using JavaScript and DOM manipulation. It features a dynamic 3×3 grid where random tiles turn black and users must click within a 10-second time limit. This game includes real-time scoring and timers, updating the interface dynamically so faster reactions earn higher points. Organized the front-end code to keep the interface clear and maintainable.

## Tech Stack
- **Language:** JavaScript, HTML, CSS  
- **Concepts:** DOM Manipulation

## Features
- Dynamic 3x3 grid
- Random tile activation
- Reaction based scoring system
- 10 second time limit

## How to Run
- Open `index.html` in your browser
- Start clicking black tiles before the time runs out
